 Формат команды: [-команда] [значение]
 Доступны следующие команды:

 -n - название ленты;
 -a - адрес RSS;
 -p - командный файл для помещения сообщения в почтовую базу;
 -o - сообщение из RSS ленты, сформированное программой;
 -t - шаблон по которому будет создано сообщение;
 -l - language file.

 Можно использовать любые параметры из config file

 Стандартные примеры:
 rss2fido [адрес_ленты]
 rss2fido -c [config_file]
 (так же смотрите в _example[XXX].bat)
