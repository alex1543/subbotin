Используйте конфигурационный файл rss2fido
в случае необходимости.
С помощью конфиг файла можно задать
пути к системным файлам программы,
а также программа может работать
с несколькими RSS лентами.

Чтобы подключить конфиг файл к программе
в коммандной строке введите следующее:
rss2fido.exe -c
Программа rss2fido найдет первый конфиг.
файл по названию в тек. каталоге.
Также можно указать путь явно, например так:
rss2fido.exe -c d:\fido\rss2fido\config.cfg
