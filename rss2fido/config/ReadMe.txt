Use configuration file rss2fido if necessary.
The config file you can specify the path to system files,
as well as program can work with several RSS feeds.

To use configuration file in command line enter:
rss2fido.exe -c
The rss2fido will be processed first name
of the configure file with the expansion cfg
in the directory with the program.
You can also point the way clearly, such as:
rss2fido.exe -c d:\fido\rss2fido\config.cfg
