format commands: [-commands] [variable]
 The following commands:

 -na - task Name;
 -ad - address RSS;
 -po - command file for posting messages in the mailbox;
 -ou - message from the RSS formed programme;
 -te - template which will be a message;
 -la - language file.

 You can use any variable from the configuration file

 Standard examples:
 rss2fido [address_RSS]
 rss2fido -c [configuration_file]
 (just see _example[XXX].bat)
