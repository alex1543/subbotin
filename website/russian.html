Russian page in the rss2fido by_Rain 27/09-2006g.
Эта программа предназначена для отслеживания новых сообщений в RSS и помещения специально сформированного сообщения в Фидонет.

  Преимущества перед другими программами и скриптами:
  1) настраиваемый шаблон будущего сообщения;
  2) поддержка нескольких кодовых страниц;
  3) четыре варианта запуска программы:
  а) без параметров;
  б) с указанием одного лишь адреса RSS;
  в) с указанием большинства параметров из командной строки с помощью ключей
  (подробнее смотрите rss2fido --help);
  г) работа с конфигурационным файлом
  (для этого используется ключ -c path_config_file).
  4) быстрая работа программы за счет обработки данных в оперативной памяти;
  5) настраиваемый language file, который позволяет:
  а) поддерживать много языков;
  б) использовать макросы для настройки цвета, внешнего вида, запуска приложений,
  ведения файла истории и многого другого.
  6) настраиваемый config file позволяет:
  а) задавать неограниченное колличество задач для формирования исходного сообщения;
  б) имеет множество настроек.
  7) Три результата работы программы:
  а) создание сформированного сообщения в виде файла;
  б) помещение сообщения в почтовую базу: squish, jam, msg;
  в) создание почтового пакета (pkt).
  8) Создавать файл в UUE.
  9) Возможность отправки файлов в Bink Style Outbound.
  10) Встроенный язык макросов.
  11) Возможность отслеживания новых сообщений.
  12) Работа программы через прокси сервер.
  13) template file позволяет записать сообщение в исходный файл так,
  как у Вас хватит фантации. :)
